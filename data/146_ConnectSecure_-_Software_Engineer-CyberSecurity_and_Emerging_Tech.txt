DETAILED JOB DESCRIPTION
Software engineer
We are seeking a highly motivated and detail-oriented Software Engineer to join our dynamic team. In this role, you will be responsible for building, maintaining, and monitoring high-performance systems that support our cutting-edge projects. We prioritize applicants with experience in working with GPUs, Vector Databases, and Retrieval-Augmented Generation (RAG) systems. Key areas of focus include threat detection, compliance, system hardening, and automation of security protocols


Key Responsibilities:
Design, Development, and Maintenance:
● Assist in designing, developing, and maintaining high-performance systems, with a focus on managed services.
● Work on both front-end and back-end development, utilizing Python for back-end and React for front-end.
● Help monitor and optimize system performance, security, and scalability.
Front-End Development:
● Support the design and development of responsive, visually appealing user interfaces using React.
● Assist in optimizing applications for maximum speed and scalability.
● Collaborate with UX/UI designers to translate designs and wireframes into high-quality code.
Back-End Development:
● Develop and maintain back-end components using Python under the guidance of senior engineers.
● Assist in creating and maintaining APIs to ensure a seamless flow of data between the server and users.
● Contribute to server-side security and data protection.
Database Management:
● Learn to design and manage database schemas using PostgreSQL.
● Assist in implementing data storage solutions and optimizing database performance.
● Exposure to Vector Databases and GPU-accelerated workloads is a plus.
Adapter Development:
● Participate in creating adapters that enable AI systems to connect to various data sources efficiently.
Testing and Maintenance:
● Contribute to developing and maintaining unit and end-to-end tests to ensure code integrity and application quality.
● Assist in regularly updating applications to improve performance and address security vulnerabilities.
Collaboration and Project Management:
● Work closely with project managers and cross-functional teams to understand requirements and deliver software solutions that meet business goals.
● Participate in code reviews and share knowledge with team members to enhance software quality and effectiveness.
Technical Skills Required:
● Basic understanding of client-side scripting and JavaScript frameworks, particularly React.
● Foundational knowledge of Python for server-side programming.
● Familiarity with PostgreSQL and a basic understanding of NoSQL databases.
● Exposure to development tools such as Babel, Webpack, and NPM.
● Familiarity with version control tools, particularly Git.


Qualifications:
● Strong willingness to learn and adapt to new technologies and methodologies.
● Good organizational and communication skills.
