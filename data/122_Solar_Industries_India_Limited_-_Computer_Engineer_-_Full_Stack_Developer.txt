Key Responsibilities
• Software Development: Develop clean, efficient, and reliable code primarily using C and C++ in a Linux environment.
• Front-End Development Support: Assist in developing user interfaces using the Qt framework.
• Database Management: Engage with SQL databases for data storage and management requirements.
• System Integration & Validation: Participate in the integration, testing, and validation of software modules within the overall system architecture.
• Debugging and Problem Solving: Utilize strong analytical skills to debug software issues, troubleshoot problems, and ensure robust system performance.
• Device Driver Knowledge: Apply foundational knowledge of device drivers in a Linux operating system context.
• Documentation and Collaboration: Work collaboratively with the engineering team, providing clear documentation of processes and results.
