DETAILED JOB DESCRIPTION
Relevant Experience Required:
1. 3+ years of Experience in CFD/CAE simulations
2. Good knowledge in Starccm+ and Abaquas tools
3. Strong knowledge of FEM, CFD theory
4. Practical experience in developing, training, validating, and deploying machine learning models using PyTorch/ TensorFlow/Jax.
5. Strong understanding of core machine learning concepts, statistics, and data analysis techniques.
6. Proficiency in Python programming with strong problem-solving and algorithmic thinking skills applied to AI/ML development.
7. Experience in deploying AI-assisted simulation tools in production environments, including version control, CI/CD, and containerization (Docker, MLflow, or similar).
8. Strong background in scientific machine learning (SciML) techniques such as reduced-order modeling, surrogate modeling, and uncertainty quantification for engineering simulations.
9. Familiarity with integrating AI models into CAE workflows—for example, embedding predictive models into STAR-CCM+, Abaqus, or custom pre/post-processing pipelines using Python or Java macros.
10. Familiarity with Physics-Informed Neural Networks (PINNs)/Operator learning in physics or hybrid physics–ML approaches, experience using libraries like DeepXDE, NVIDIA Modulus.
11. Strong communication skills (German Language proficiency a plus)
12. Experience in developing novel Simulation methods would be a plus.
