DETAILED JOB DESCRIPTION
As a Data Analyst, you are required to:


• Develop prototypes and proof of concepts using multiple data-sources, AI/ML methods and big-data technologies
• Port data across data-sources /DB systems in order to improve performance, scalability or to facilitate analytics.
• Stay curious and enthusiastic about using related technologies to solve problems and enthuse others to see the benefit in business domain
Qualification: Bachelor's or Master's in Computer Science & Engineering, or equivalent. Professional Degree in Data Science, Engineering is desirable.

Desired Knowledge & Experience:
• Sound knowledge in programming languages like Python, R will be an added advantage
• Strong written/communication skill to be work with partners along the globe
• Strong theoretical knowledge of probability, statistics, and linear algebra. Should have done coursework on these topics (at least 1 semester) in their Engineering.
• Should have done coursework on AI, ML algorithms in their Engineering.
• Should have done a coursework on Python programming (both theory and lab sessions). Usage of atleast one Data Science library, working experience with data frames.
• Should have interest and passion to work on Data Science (as compared to full stack / or regular development projects).
• Having prior experience in developing mini-projects, POC on AI/ML based topics is desirable.
• Having prior experience as interns on data science projects is desirable.
• Practical knowledge on building data visualization tools is desirable.
• Knowledge of any reporting/BI tool (e.g. Qlik, Tableau, PowerBI, SAS VA etc…)
• Coursework on databases, big data, data warehouse is desirable.
• Knowledge / Experience in conventional DB technologies (RDBMS, Object Oriented)
• Strong Knowledge/Experience in using query languages such as SQL
• Basic knowledge in Data Warehousing concepts (Data Modelling, STAR Schema, Snowflake Schema and various Data Warehousing concepts)
• Basic knowledge on Extract Transform & Load concept (any tool is fine e.g. SAS DI Studio, Informatica…etc)


Required Soft skills & Other Capabilities:
• Great attention to detail and good analytical abilities.
• Good planning and organizational skills
• Collaborative approach to sharing ideas and finding solutions
• Ability to work independently and also in a global team environment.
