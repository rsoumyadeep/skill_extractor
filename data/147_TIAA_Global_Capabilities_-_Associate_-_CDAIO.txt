DETAILED JOB DESCRIPTION
Position Overview
Join TIAA as a Technical Associate working across Production Operations, Data Engineering, Quality Assurance, or Enterprise Applications. This role combines hands-on technical work with cutting-edge AI/ML automation in a mission-driven organization.
What You'll Do
• Systems & Operations: Monitor cloud platforms (AWS, Snowflake), respond to incidents, support 24/7 mission-critical systems
• Data Engineering: Build ETL/ELT pipelines, move data between sources and warehouses, ensure data quality
• Quality Assurance: Write test cases, execute testing, validate data accuracy, automate test workflows
• Enterprise Applications: Configure Workday HCM modules, build reports, support integrations
• Automation: Identify repetitive tasks and automate using AI tools, scripts, and workflows
AI & Innovation Focus
• Use Generative AI tools (AWS Bedrock, Claude, Amazon Q) for code generation and documentation
• Experiment with prompt engineering and AI-powered automation
• Learn and implement AI agents for testing and operational tasks
• Explore emerging technologies like Agentic AI frameworks
Learning & Growth
• Work under mentorship of senior engineers on real production systems
• Participate in Agile teams and collaborate with data engineers, developers, and business analysts
• Document solutions and contribute to knowledge repositories
• Attend training sessions and work toward technical certifications
Technical Skills You Bring


Must Have:
• Programming: Solid Python or Java skills with basic understanding of object-oriented programming
• SQL: Ability to write queries, joins, and understand data relationships
• Linux/Unix: Comfortable with command-line environments and basic shell scripting
• Problem-solving: Strong analytical and logical reasoning abilities
Good to Have:
• Exposure to AWS services (S3, EC2, Lambda) through coursework or projects
• Understanding of databases (MySQL, PostgreSQL) or data warehouses
• Familiarity with Git/GitHub for version control
• Experience with data formats (JSON, CSV, XML)
• Use of AI coding assistants (GitHub Copilot, ChatGPT, etc.)
• Knowledge of testing concepts or test automation
• Understanding of APIs and web services
