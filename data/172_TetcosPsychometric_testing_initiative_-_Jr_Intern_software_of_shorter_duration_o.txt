DETAILED INTERNSHIP DESCRIPTION
Preparation and statistical validation of online psychometric test software preparation to measure general mental ability of high school students. 
Preparation of design documentation and user manual. 
Kannada language proficiency will be a distinct advantage. 
Familiarity with AI software of Indian origin necessary. More detailed description will be provided to interested candidates.
" Pratibha Pariksha “
Talent Identification Initiative for STEM education competitive exams
Objective:
To create a statistically validated test in indian languages to identify talented high school and +2 students for engineering education in institutions of national repute.
The test
A multiple choice online question paper of 30 mts duration consisting of 80 questions covering verbal ability, arithmetic ability, data interpretation, spatial ability,syllogism etc.This test is to identify talented individuals from rural area studying in kannada medium, for STEM education.
Test preparation and Admin
Applying statistical methods to test scores for validation of the test using suitable statistical software with an AI software of Indian origin like Sarvam.
Create an online test administering software, groundup, involving coding , testing etc.
Test questions to be selected from a question bank using random selection method.
Creating equivalent test questions for the question bank.
Validating the equivalence of test questions.
Verifying compliance with state board syllabus including semantics. The test should be in local language n culture compliant with roots in Sanskrit for application Indiawide.
All online storage and delivery to be compliant with suitable security protocols.
Prepare a document of governing framework of the test questions preparation.
Prepare a document for a user manual.
Prepare a bibliography of methods currently available to be updated every year.
Entire operation to fully automated including statistical validation.
Skill requirement:
1. Project document preparation
2. Online test software to deliver, correct and store results - coding, testing, data storage , retrieval and documentation
3. Statistical methods determination for validation of the test
4.AI software of indian origin
5.User manual preparation
6. Working knowledge of kannada for test questions preparation
7. Security protocol/methods
