# 📊 Skill Demand Analyzer

A Python-based tool to extract and analyze skill demand from text data (e.g., job descriptions, notes, or repositories).  
It identifies key skills, ranks them by frequency, and visualizes the results.

---

## 🚀 Features

- 📂 Process multiple `.txt` files
- 🧠 Extract predefined technical skills
- 📊 Generate sorted skill frequency distributions
- 📈 Visualize results using histograms
- 🌐 Interactive dashboard using Streamlit
